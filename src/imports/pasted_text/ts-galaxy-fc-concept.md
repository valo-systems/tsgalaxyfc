Create a premium, modern, fully responsive website concept for TS Galaxy FC using their existing public brand direction, current colour scheme, football identity, and “The Rockets” club personality.

The design must feel like a professional top-tier football club platform, not a generic sports template. It should be bold, mobile-first, commercial, emotional, and easy for fans to use. The website should work beautifully on mobile, tablet, desktop, and large screens.

Use the current TS Galaxy FC visual identity as the base:
- Keep the existing club colours and brand feel.
- Use a dark, premium sports foundation where appropriate.
- Use the club’s existing accent colours from the badge/website styling.
- Do not invent a completely new brand.
- Do not make it look like a different club.
- The redesign must feel like an evolution of the current TS Galaxy FC website, but much more modern, responsive, energetic, and premium.

The website should be designed for:
1. Football supporters
2. Matchday visitors
3. Merchandise buyers
4. Club members
5. Academy applicants
6. Sponsors and commercial partners
7. Media and journalists
8. Club staff who need to manage content

Main design goal:
Turn the TS Galaxy FC website into a mobile-first football platform that helps fans check fixtures, buy tickets, follow results, shop merchandise, join membership, follow the men’s team and TS Galaxy Queens, view academy opportunities, and engage with sponsors.

Design principles:
- Mobile-first layout
- Premium football club feel
- Fast, clean, responsive UI
- Strong visual hierarchy
- Clear calls to action
- Big matchday energy
- Professional sponsor presentation
- Easy content management feel
- Modern cards, badges, tabs, filters, and dashboards
- Strong use of photography, club colours, match graphics, and football textures
- Avoid clutter
- Avoid generic corporate layout
- Avoid old WordPress-style design
- Avoid tiny text and table-heavy mobile layouts

Create the following responsive pages/screens:

1. Mobile Home Page
2. Desktop Home Page
3. Match Centre Page
4. Fixtures & Results Page
5. News Listing Page

Desktop rules:

- Strong hero imagery
- Dashboard-style match information
- Multi-column content where useful
- Premium club feel

Tablet rules:
- Hybrid between mobile cards and desktop grid
- Avoid cramped navigation
